# HapiColi クラス設計

## 概要

HapiColiは、エンジン非依存で利用可能な当たり判定デバッグ・解析ツールである。

本ツールは以下の流れで動作する。

```text
記録
 ↓
保存
 ↓
解析
 ↓
問題検出
 ↓
改善提案
```

---

# クラス構成

```text
HapiColi
│
├─ HapiColiManager
│
├─ Recorder
│
├─ Analyzer
│
├─ Optimizer
│
├─ LogManager
│
├─ UIManager
│
├─ TestRunner
│
├─ TestRule
│
├─ FrameData
│
├─ TestResult
│
└─ JsonSerializer
```

---

# HapiColiManager

## ファイル

```text
HapiColiManager.h
HapiColiManager.cpp
```

## 役割

ツール全体の制御を行う。

### 管理対象

* Recorder
* Analyzer
* Optimizer
* UIManager
* LogManager

---

# Recorder

## ファイル

```text
Recorder.h
Recorder.cpp
```

## 役割

ゲームから情報を収集する。

### 記録対象

* Position
* Rotation
* Velocity
* Collider情報
* Collision結果

### 出力

FrameData

---

# Analyzer

## ファイル

```text
Analyzer.h
Analyzer.cpp
```

## 役割

記録されたデータを解析する。

### 機能

* Happy判定
* Unhappy判定
* 成功率計算
* 失敗率計算
* 問題フレーム抽出

# Optimizer

## ファイル

```text

Optimizer.cpp
```

* Collider拡大

* Raycast利用

# FrameData

## ファイル

```text
FrameData.h
```

## 役割

1フレーム分の記録データを保持する。

### 例

```cpp
struct FrameData
{
    Vector3 position;
    Vector3 velocity;

    bool hit;
};
```
